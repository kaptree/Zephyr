实现任务归档查询页面，包含多条件筛选栏、时间轴/卡片双视图切换、归档导出功能。

- 依赖任务05（核心组件）完成

- 遵循文档第9.3节归档与追溯设计

## 具体步骤

### 7.1 页面路由

- 路由路径：`/workbench/archive`

### 7.2 筛选栏 (`src/pages/ArchivePage.vue` 顶部)

```
┌──────────────────────────────────────────────────────────┐
│ [时间范围: 2024-01-01 ~ 2024-12-31] [标签: ▼] [人员: ▼] │
│ [部门: ▼] [关键词: _______] 🔍搜索  [清空]               │
└──────────────────────────────────────────────────────────┘
```

- 时间范围：日期范围选择器（可使用 DaisyUI input[type="date"]）
- 标签：复用 TagSelector 组件
- 人员：复用 UserPicker 组件
- 部门：下拉选择器
- 关键词：文本输入框 + 搜索按钮

### 7.3 结果展示区

#### 视图切换按钮

- "时间轴" / "卡片" 切换按钮组
- 当前选中态使用交互蓝高亮

#### 时间轴视图

```
── 2024年12月 ──────────
  📅 12-15  │ ┌─────────────────┐
            │ │ [已归档] 任务卡片  │
            │ └─────────────────┘
  📅 12-10  │ ┌─────────────────┐
            │ │ [已归档] 任务卡片  │
            │ └─────────────────┘
── 2024年11月 ──────────
  📅 11-28  │ ┌─────────────────┐
            │ │ [已归档] 任务卡片  │
            │ └─────────────────┘
```

- 以月分组
- 左侧日期 + 竖线时间轴
- 右侧卡片（复用 StickyNoteCard，添加"已归档"水印角标）

#### 卡片视图

- 复用 StickyNoteCard 组件
- 每张卡片右下角显示"已归档"半透明水印 + 归档时间
- 使用与工作台一致的 CSS Grid 布局

### 7.4 卡片操作

每张归档卡片支持：

- **查看详情**：打开详情侧滑面板（只读模式）
- **导出Word**：调用 `POST /api/v1/notes/{id}/export`，显示进度，完成后下载
- **恢复至工作台**：调用 `POST /api/v1/notes/{id}/restore`，恢复后从归档列表移除
- **批量导出**：选中多张卡片，批量导出 Excel/Word

### 7.5 分页加载

- 使用 serve r端分页
- 触底自动加载更多（Intersection Observer）
- 显示"已加载 X / 共 Y 条"

### 7.6 数据权限隔离

- 普通用户：仅查看自己的归档
- 组长：查看本组归档
- 部门管理员：查看本部门归档
- 系统管理员：查看全部归档
