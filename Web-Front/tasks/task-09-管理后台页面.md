## 任务目标
实现4个管理后台页面：部门库管理、人员库管理、标签库管理、模板库管理。

- 依赖任务05（核心组件）完成
- 依赖任务10（权限系统）完成

## 具体步骤

### 9.1 部门库管理 (`src/pages/admin/DepartmentsPage.vue`)

#### 页面布局
- 左侧：树形部门结构（可展开/折叠、拖拽排序）
- 右侧：部门详情编辑区（选中部门后显示）

#### 功能
- **查看部门树**：调用 `GET /api/v1/departments`
- **新建部门**：在选中部门下创建子部门
- **编辑部门**：名称修改
- **删除部门**：确认对话框 + 级联提示
- **拖拽排序**：调整部门层级

#### 权限
- 系统管理员：全部操作
- 部门管理员：仅操作本部门及下属

### 9.2 人员库管理 (`src/pages/admin/UsersPage.vue`)

#### 页面布局
- 顶部：搜索框 + 部门/角色筛选
- 主体：人员列表（DaisyUI table）
- 分页栏

#### 功能
- **搜索**：支持姓名/拼音首字母
- **筛选**：按部门、角色、状态
- **新建人员**：弹窗表单（姓名、部门、角色、账号、密码）
- **编辑人员**：修改信息、角色变更
- **角色分配**：super_admin / dept_admin / group_leader / user / screen_role
- **停用/启用**

#### API 调用
- 列表：`GET /api/v1/users`
- 创建：`POST /api/v1/users`
- 更新：`PUT /api/v1/users/{id}`
- 删除：`DELETE /api/v1/users/{id}`

### 9.3 标签库管理 (`src/pages/admin/TagsPage.vue`)

#### 页面布局
- 顶部：作用域切换（个人/系统/全部）+ 新建按钮
- 主体：标签卡片网格（彩色胶囊展示）

#### 功能
- **查看标签**：调用 `GET /api/v1/tags`
- **新建标签**：弹窗（名称、颜色选择、分类、作用域）
- **编辑标签**：名称、颜色修改
- **删除标签**：确认对话框
- **颜色选择器**：预设12种语义色

#### 标签颜色预设
```
#EF4444 红    #F97316 橙    #EAB308 黄    #22C55E 绿
#14B8A6 青    #3B82F6 蓝    #8B5CF6 紫    #EC4899 粉
#78716C 棕    #64748B 灰蓝  #94A3B8 灰    #475569 深灰
```

### 9.4 模板库管理 (`src/pages/admin/TemplatesPage.vue`)

#### 权限
- 仅系统管理员可访问
- 路由守卫 + v-permission 双重保护

#### 页面布局
- 左侧：模板分类列表
- 右侧：模板详情/编辑器

#### 功能
- **查看模板**：`GET /api/v1/templates`
- **新建模板**：模板名称 + 字段配置 + 布局选择
- **编辑模板**：字段增删改 + 排序
- **模板字段类型**：文本、日期、下拉、多选、富文本
- **删除模板**

#### 数据上报类型模板
默认内置以下模板（公安业务场景）：
- 工作日报模板
- 情报研判模板
- 会议纪要模板
- 案件协查模板

### 9.5 通用管理页面布局

- 使用 DaisyUI Table 组件（`table table-zebra`）
- 分页使用 DaisyUI 分页组件
- 新建/编辑弹窗使用 DaisyUI Modal
- 删除确认使用 DaisyUI Alert/Confirm
- 空态提示："暂无数据"

## 验收标准
1. 部门树正常加载、展开/折叠、拖拽排序
2. 人员列表支持搜索、筛选、分页
3. 人员新建/编辑/角色分配流程完整
4. 标签卡片网格正确展示，颜色选择器可用
5. 模板字段编辑器功能正常
6. 超管专属路由其它角色不可见/不可访问
7. 所有表单验证正确
